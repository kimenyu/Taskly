About Taskly

Taskly was inspired by our personal struggles with managing tasks efficiently. As ALX students, we realized the need for a powerful to-do list app that could help us stay organized with our coursework and personal projects.This web application is a testament to how we have honed and improved our skills through the ALX software engineering program.

This project is a portfolio project for ALX Software Engineering Program. You can learn more about ALX here.
